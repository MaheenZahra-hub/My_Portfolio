# Personal Portfolio Website

Welcome to the repository for my personal portfolio website. This project serves as a digital space to showcase my front-end development skills, highlight my engineering projects, and share my journey as a computer science student.

## What's Inside

The website provides a clean, user-friendly interface that outlines my technical background and practical work:

* **Project Showcase:** Direct links and descriptions of my core engineering projects, including database optimization models and academic repositories.
* **Technical Skill Set:** A breakdown of the programming languages, database systems, and design concepts I work with.
* **About Me:** A brief overview of my educational background, core development interests, and aspirations.

## Technologies Used

* **Front-End:** HTML5, CSS3, and JavaScript (ES6+) for layout, styling, and interactive elements.
* **Design & Layout:** Built with a strong focus on responsive design to ensure a seamless layout across both mobile and desktop screens.
